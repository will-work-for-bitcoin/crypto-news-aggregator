# Crypto News Aggregator

Crypto news aggregator with sentiment analysis. Pure Python, zero dependencies.

## Installation

```bash
git clone https://github.com/will-work-for-bitcoin/crypto-news-aggregator.git
cd crypto-news-aggregator
```

## Features

- 🔹 Multi-source RSS feeds
- 🔹 Sentiment scoring
- 🔹 Filter by topic
- 🔹 Pure Python 3.7+, zero dependencies
